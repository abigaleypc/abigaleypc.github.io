Posted on 2017-03-23 | In Front-End |

项目目录

初始化项目

在命令面板敲下npm init 初始化项目并生成 package.json 。

  • 新建项目 mkdir angular-webpack
  • 切换进入 cd angular-webpack
  • 初始项目 npm init
  • 安装框架 npm install angular --save-dev ( 如已安装yarn，可直接执行yarn add angular )

入口文件

  • 新建src文件夹，存放业务代码。在src下新建index.html入口文件。

    src / index.html

  • 新建app.js文件，与index.html实现双向绑定。

src / app.js

添加Webpack模块

  • 安装模块 npm install webpack --save-dev

编写配置文件

  • 新建配置文件夹 webpack.config,在该文件夹之下新建base.js

webpack.config / base.js

webpack.config / dev.js

webpack.config / pro.js

打开命令面板
