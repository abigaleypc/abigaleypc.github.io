JS的组成：ECMAScript、BOM和DOM

Posted on 2017-03-09

Javascript由三部分组成：

  • ECMAScript:javaScript的核心
  • DOM:规定了访问HTML和XML的接口
  • BOM:提供了独立于内容而在浏览器窗口之间进行交互的对象和方法

ECMAScript

这部分不属于浏览器，它可以定义变量，函数等

DOM

全称:Document Object Model , 文档对象模型

DOM定义了操作HTML的接口，比如写了一个html文档，浏览器根据DOM元素渲染成树形结构

BOM

全称:Browser Object Model , 浏览器对象模型

BOM 定义了 JavaScript 操作浏览器的接口，提供了访问某些功能（如浏览器窗口大小、版本信息、浏览历史记录等）的途径以及操作方法。
